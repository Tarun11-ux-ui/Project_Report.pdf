# CryptoCurrency AI Dashboard Analysis

## Project Overview
A cryptocurrency analysis dashboard built using Microsoft Excel.

## Features
- Time-Based Visibility & Sentiment Analysis
- Crypto Coin Comparison Dashboard
- Liquidity Analysis Dashboard
- Price Change Trend & Alerts
- Historical Price Change Insights
- Low Budget Investment Insights

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Lookup Functions
